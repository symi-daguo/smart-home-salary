# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| main    | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability, please email **security@example.com** with the details. We take all security reports seriously and will respond as quickly as possible.

When reporting, please include:
- A description of the vulnerability
- Steps to reproduce
- Affected versions/commits
- Any relevant logs or screenshots

## Disclosure Process

1. We will acknowledge receipt of your report within **72 hours**.
2. We will work to reproduce and assess the impact.
3. We will develop and test a fix.
4. We will coordinate a release and publish a security advisory.

Please **do not create a public issue** for security reports. We appreciate responsible disclosure.
